# Custom-news

Custom-news es un repositorio para la ingesta y procesado de noticias personalizadas.

Funcionalidades principales:
- Recolección de fuentes RSS y APIs de noticias
- Normalización y filtrado de contenido
- Pipeline de análisis y extracción de entidades
- Generación de resúmenes y alertas

Estructura del repositorio:
- data/ : scripts y ejemplos de datos
- scripts/ : utilidades para ingestión y procesamiento
- models/ : artefactos y modelos usados

Cómo contribuir:
1. Abre una issue describiendo el cambio propuesto.
2. Crea una rama siguiendo el prefijo `openclaw/` para cambios automáticos o `feature/` para mejoras manuales.
3. Envía un PR con pruebas y descripción clara.
